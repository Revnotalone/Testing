# Telegram Relay Bot

Bot Tiruan -> Master Bot relay menggunakan Telethon + python-telegram-bot.

## Cara Deploy
1. Buat project baru di Railway.
2. Tambahkan repo GitHub ini.
3. Isi **Variables** di Railway:
   - `API_ID`
   - `API_HASH`
   - `TOKEN_TIRUAN`
   - `MASTER_USERNAME`
   - `SESSION_NAME` (default: buyer_session)
